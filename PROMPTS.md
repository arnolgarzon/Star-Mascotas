# PROMPTS.md

## 📌 Uso de Inteligencia Artificial en el desarrollo

Durante el desarrollo de este proyecto utilicé IA como apoyo para acelerar la construcción, mejorar la calidad del código y aplicar buenas prácticas de desarrollo.

A continuación, se presentan los 3 prompts más relevantes utilizados:

---

## 1️⃣ Generación de arquitectura base del proyecto

### Prompt:
Actúa como un Desarrollador Senior. Genera la estructura de carpetas y el boilerplate inicial para un sistema de inventario sencillo usando FastAPI + React + SQLite. Necesito que el backend tenga un CRUD básico de productos (Nombre, Categoría, Precio, Stock) y el frontend una tabla para visualizarlos. Sigue principios de código limpio y separación de responsabilidades.

### ¿Por qué lo usé?
Este prompt me permitió:
- Definir una arquitectura clara desde el inicio
- Separar correctamente frontend y backend
- Aplicar buenas prácticas como modularidad y organización por capas

Esto evitó retrabajo y aseguró una base sólida para el desarrollo.

---

## 2️⃣ Construcción del frontend con buenas prácticas

### Prompt:
Ayúdame a crear el frontend utilizando código limpio, escalable y bien comentado (inglés y español). Necesito un formulario de productos y una tabla para visualizarlos, además de un botón para exportar datos a CSV sin usar librerías externas pesadas.

### ¿Por qué lo usé?
Este prompt fue clave para:
- Implementar un frontend funcional con React
- Mantener código legible y mantenible
- Evitar dependencias innecesarias (optimización)
- Agregar funcionalidad útil como exportación CSV

---

## 3️⃣ Mejora de UI/UX y mantenibilidad

### Prompt:
Optimiza el diseño para que sea responsivo, use una paleta profesional acorde a una tienda de mascotas y mejora la mantenibilidad usando un sistema de variables globales (theme.css). Añade validaciones visuales y mejora la experiencia de usuario.

### ¿Por qué lo usé?
Este prompt permitió:
- Mejorar significativamente la experiencia de usuario (UX)
- Implementar diseño responsive (mobile + desktop)
- Centralizar estilos para facilitar mantenimiento
- Dar un acabado más profesional al sistema

---

## 🎯 Conclusión

La IA fue utilizada como una herramienta de apoyo para:
- Acelerar el desarrollo
- Mejorar la calidad del código
- Aplicar buenas prácticas

El enfoque siempre fue comprender, adaptar y validar cada solución, asegurando que el resultado final fuera coherente y funcional.
